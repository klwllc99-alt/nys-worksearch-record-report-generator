Help and user guide

This app converts work-search records from a CSV or XLSX file into a WS5-ready PDF. You can review, correct, sort, preview, and then download your final output.

Back to app Download CSV template Download XLSX template

Quick start

  1. Download the template if you need a starter file.
  2. Upload a CSV or XLSX file, or drag and drop it into the app.
  3. Review the editable grid and fix any rows.
  4. Enter optional claimant details like name, NYS ID, and SSN last 4.
  5. Choose whether you want one merged PDF or a ZIP with one PDF per week.
  6. Preview the result, then generate your download.

Main features

File import

Supports both CSV and XLSX uploads. The app parses your rows and loads them into the grid automatically.

Editable record grid

Every row can be corrected inline. You can also add new rows, clear data, or sort by column.

Clipboard paste

Paste spreadsheet rows directly if copying from Excel, Google Sheets, or another table.

PDF preview

Render a preview before final download so you can confirm the information looks right.

Overflow handling

Enable the option to generate all activities when your work-search history extends beyond the first WS5 page.

Admin dashboard

The admin area includes login, user management, and request metrics for monitoring app activity.

Record types explained

  • employer_contact: Use for job applications, calls, emails, interviews, and employer outreach.
  • other_activity: Use for workshops, career center visits, resume updates, and similar search activities.

Make sure each row has a valid date and enough detail to describe the work-search action clearly.

Troubleshooting tips

  • If the upload is rejected, confirm the file has the expected columns and valid dates.
  • If clipboard paste is blocked by the browser, click the paste box and use Ctrl+V manually.
  • If your output spans multiple weeks, choose the per-week ZIP mode for separate files.
  • If there are more activities than fit on one form, turn on the overflow generation option.
